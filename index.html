Inventario

Ventana donde podrá visualizar el listado de inventario actual

Los campos requeridos fueron señalados con el tipo de dato esperado. Ingrese solamente ese tipo de dato, no ingrese datos repetidos

  • Agregar Producto Sirve para agregar productos, para ello ingrese la información en los valores requeridos. Use el ejemplo que aparece en cada uno de ellos para ingresar los datos de manera correcta. De no ser así, el sistema le alertará, y no permitará realizar el producto. Presione el botón luego de llenar todos los campos
  • Modificar Producto Seleccione el nombre del producto que desea modificar, posteriormente modifique los campos deseados, presione el botón luego
  • Eliminar Producto Seleccione el nombre del producto que desea eliminar, luego presione el botón, clic en aceptar
  • Limpiar Vaciará los datos ingresados
Actualizado hace un mes.

Facturar

Ventana donde se genera la factura de un cliente. Siga los pasos para realizar una venta exitosa

Seleccione el cliente al que le desea facturar, luego agregar o escanear los productos que desea agregar a la factura y finalmente generar la factura. Para envios complete con la informacion adicional para el envio.

  • Nuevo Cliente Sirve para acceder de manera rapida al agregar un nuevo cliente
  • Cancelar venta Elimina los datos de factura
  • Generar Venta Genera la factura, posteriormente la imprime
  • Actualizar Informacion Actualiza la informacion de venta, use este boton si no han habido cambios en el total de la factura despues de ingresar productos. Esto no es necesario, el sistema actualizará la información al momento de realizar la venta, sin embargo, de no ser así, presione el boton
  • Escanear Codigo Abre una ventana para escanear el codigo del producto, es la forma con la que puede agregar producto a la venta
  • Ayuda Muestra una ventana de ayuda para el ususario
Escanear Codigo

Ventana para escanear codigo de barra de un producto

Debe escanear o ingresar el codigo de bara del producto que dese agregar a la factura

  • Escanear Escanea el codigo de barra del producto. Solamente ingrese el codigo de barra, no otro otro tipo de dato
  • Buscar desde archivo Busca en el almacenamiento del programa para agregar el codigo de barra de un producto
  • Agregar Agrega el producto escaneado o seleccionado a la factura. Despues de este boton, los datos se actualizarán, y permitirá seleccionar la cantidad de producto a agregar. Click despues en el boton que aparecerá, y el producto se agregará automaticamente en el carrito de venta
Actualizado hace dos meses
